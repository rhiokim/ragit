title: config
description: Configure repository-level RAGit behavior
---

The `config` family controls repository-local RAGit settings.

## Choose A Subcommand

- Use <a href="/en/docs/commands/config/set">`config set`</a> to update one config key.

## Family Boundary

- `config` changes behavior.
- `status` and `doctor` inspect behavior.
- `init` prepares defaults but does not replace targeted config edits.
