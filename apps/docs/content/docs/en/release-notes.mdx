## v0.1 Highlights

- Full CLI command surface for MVP
- 8 document types including SPEC/PB
- `SPEC` documents implementation-level functional requirements and contracts
- `PB` documents implementation phase topology and binding architecture
- Interactive `init` with AGENTS parsing
- Hybrid retrieval and context pack
- sqlite-vss compatibility migration path

## Verification

- Unit tests for doc type, masking, hybrid score
- Integration tests for incremental ingest and init flows
- Build/test checks pass on CI baseline
