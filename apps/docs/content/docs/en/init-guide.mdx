title: Init Guide
description: Interactive AGENTS-aware initialization flow
---

`ragit init` runs a 6-step interactive flow by default.

1. Environment check (and optional `git init`)
2. Mode confirmation
3. Load/create root `AGENTS.md`
4. Confirm docs template scope (ADR/PRD/SRS/SPEC/Plan/DDD/Glossary/PB)
5. Build `.ragit/guide` incrementally
6. Print summary + next actions

`SPEC` templates are used for implementation-level functional specifications, while `PB` templates describe phase topology and binding structure.

## Non-interactive

```bash
ragit init --yes
ragit init --yes --git-init
ragit init --yes --output json
```

## Generated structure

```text
.ragit/
  guide/
    guide-index.json
    templates/
```
