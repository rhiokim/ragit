## Requirements

- Node.js 20+
- pnpm 10+
- Git repository

## Install

```bash
pnpm install
pnpm build
pnpm ragit --help
```

Inside a repository checkout, run CLI commands with `pnpm ragit <command>`.

## Initialize RAGit

```bash
pnpm ragit init
pnpm ragit hooks install
pnpm ragit ingest --all
```

## Run docs locally

```bash
pnpm docs:dev
```

Then open `/en/docs` and `/ko/docs`.
