---

## Requirements

- Node.js 20+
- pnpm 10+
- Git repository

## Install

```bash
pnpm install
pnpm build
```

## Initialize ragit

```bash
ragit init
ragit hooks install
ragit ingest --all
```

## Run docs locally

```bash
pnpm docs:dev
```

Then open `/en/docs` and `/ko/docs`.
