## 요구 사항

- Node.js 20+
- pnpm 10+
- Git 저장소

## 설치

```bash
pnpm install
pnpm build
```

## ragit 초기화

```bash
ragit init
ragit hooks install
ragit ingest --all
```

## 문서 사이트 로컬 실행

```bash
pnpm docs:dev
```

브라우저에서 `/en/docs` 및 `/ko/docs`를 확인합니다.
