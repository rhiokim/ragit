## 요구 사항

- Node.js 20+
- pnpm 10+
- Git 저장소

## 설치

```bash
pnpm install
pnpm ragit --help
```

저장소 체크아웃 안에서는 CLI 명령을 `pnpm ragit <command>` 형태로 실행합니다.
`pnpm build`는 로컬 저장소 사용에는 선택 사항이며, `dist/` 번들을 직접 생성하거나 검증할 때만 필요합니다.

```bash
pnpm build
```

## RAGit 초기화

```bash
pnpm ragit init
pnpm ragit hooks install
pnpm ragit ingest --all
```

여기까지 하면 커밋 결속 검색은 바로 사용할 수 있습니다. `memory wrap`, `memory recall`, `memory promote`가 왜 분리되어 있는지 이해하고 싶다면, 프로젝트에 memory 워크플로를 넣기 전에 <a href="/ko/docs/memory-model">Memory Model Guide</a>를 먼저 읽으세요.
세부 구문과 명령별 예시가 필요하시면 <a href="/ko/docs/commands">Commands</a>로 이어서 보시면 됩니다.

## 문서 사이트 로컬 실행

```bash
pnpm docs:dev
```

브라우저에서 `/en/docs` 및 `/ko/docs`를 확인합니다.
