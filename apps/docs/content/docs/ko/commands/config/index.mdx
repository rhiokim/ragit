`config` 패밀리는 저장소 로컬 RAGit 설정을 다룹니다.

## 어떤 하위 명령을 선택하는가

- <a href="/ko/docs/commands/config/set">`config set`</a>: 하나의 설정 키를 업데이트합니다.

## 패밀리 경계

- `config`는 동작을 바꿉니다.
- `status`와 `doctor`는 동작을 관찰합니다.
- `init`은 기본값을 준비하지만 세부 설정 수정 자체를 대체하지는 않습니다.
