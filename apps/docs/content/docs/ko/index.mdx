`ragit`은 git 기반 AI Agent 워크플로를 위한 로컬 우선 RAG CLI입니다.
프로젝트 문서/컨텍스트를 수집하고 commit SHA와 결속된 스냅샷 검색을 제공합니다.

## 제공 기능

- 커밋 결속 스냅샷
- 하이브리드 검색(벡터 + 키워드)
- 에이전트 컨텍스트 패킹
- 비밀정보 마스킹 수집
