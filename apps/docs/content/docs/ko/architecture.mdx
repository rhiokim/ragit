## 런타임

- TypeScript / Node.js CLI
- 로컬 임베딩 어댑터
- `.ragit/` 하위 manifest + chunk 저장소

## 검색

- 청크 임베딩 + 키워드 점수 결합
- 가중 하이브리드 랭킹(`alpha`)
- snapshot SHA 기반 시점 조회

## 결속 모델

- 각 스냅샷은 commit SHA와 결속
- `--since <sha>` 증분 인덱싱 지원
- `post-commit`, `post-merge` 훅 자동 동기화
