## v0.1 하이라이트

- MVP CLI 명령 세트 구현
- SPEC/PB 포함 8개 문서 타입 지원
- `SPEC`로 구현체 수준 상세 기능 요구사항과 계약을 구조화
- `PB`로 구현체의 위상 구조와 결속 아키텍처를 문서화
- AGENTS 파싱 기반 인터랙티브 `init`
- 하이브리드 검색 및 컨텍스트 패킹
- sqlite-vss 호환 마이그레이션 경로 제공

## 검증

- 문서 타입/마스킹/검색 점수 단위 테스트
- 증분 인제스트 및 init 통합 테스트
- 빌드/테스트 기준 통과
