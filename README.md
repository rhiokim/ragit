# ragit
RAG for git

- 토큰 사용을 줄인다.
- 더 정교한 에이전트의 동작을 보장한다.

## Concept

- RAG 를 저장소에 함께 구축한다.
- sqlite-vss 를 이용한다.

```
- domain-drivien-design
- twelve-factor-app
- software-requirements-specification
- product-requirements-documents
- architecture-decision-records
- keep-it-simpe-stupid
- test-driven-development
```
