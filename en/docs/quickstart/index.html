    Quickstart

    Get started with RAGit and docs workflow

    Requirements

    • Node.js 20+
    • pnpm 10+
    • Git repository

    Install

    Inside a repository checkout, run CLI commands with pnpm ragit <command>. pnpm build is optional and only needed when you want to generate the dist/ bundle locally.

    Initialize RAGit

    At this point you can start using commit-bound retrieval. If you want the philosophy behind memory wrap, memory recall, and memory promote, read Memory Model Guide before you add memory workflows to your project. If you want detailed command syntax and per-command examples, continue with Commands.

    Run docs locally

    Then open /en/docs and /ko/docs.
