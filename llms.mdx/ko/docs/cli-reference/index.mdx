# CLI 레퍼런스



상세 CLI 레퍼런스는 이제 <a href="/ko/docs/commands">Commands</a>로 이동했습니다.

이 페이지는 기존 링크와 북마크를 깨지 않기 위한 호환 경로만 제공합니다.

지금 어디로 가면 되는가 [#지금-어디로-가면-되는가]

* 명령군과 leaf 명령 상세는 <a href="/ko/docs/commands">Commands</a>를 보시면 됩니다.
* 에이전트 안전 호출 규칙은 <a href="/ko/docs/agent-cli">Agent CLI Contract</a>를 보시면 됩니다.
* `memory wrap`, `memory recall`, `memory promote`의 설계 철학은 <a href="/ko/docs/memory-model">Memory Model Guide</a>에서 확인하실 수 있습니다.
