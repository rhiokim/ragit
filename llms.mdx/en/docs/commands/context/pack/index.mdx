# context pack



What It Does [#what-it-does]

`context pack` turns retrieved knowledge into a bounded packet for a specific goal. It is the command to use when the next consumer is an agent prompt rather than a person reading search results.

When To Use / When Not To Use [#when-to-use--when-not-to-use]

When to use it [#when-to-use-it]

* You have a goal and want a compact context handoff.
* You need token budgeting.
* You want to pass structured input to an agent reliably.

When not to use it [#when-not-to-use-it]

* You only want to inspect raw retrieval hits. Use <a href="/en/docs/commands/query">`query`</a>.
* You want to resume active work with open loops and working state. Use <a href="/en/docs/commands/memory/recall">`memory recall`</a>.

Syntax [#syntax]

```bash
pnpm ragit context pack [goal] [--input <path|->] [--budget <tokens>] [--at <sha>] [--view minimal|default|full] [--format text|json|both]
```

Arguments And Options [#arguments-and-options]

* `goal`: The goal that should drive context selection.
* `--input <path|->`: Provide a JSON payload with `goal`, optional `budget`, and optional `at`.
* `--budget <tokens>`: Token budget for the selected packet.
* `--at <sha>`: Use a specific snapshot instead of the current HEAD state.
* `--view minimal|default|full`: Control how much detail is projected into the packet.
* `--format text|json|both`: Choose text, JSON, or both.
* `--cwd <path>`: Run against another repository.

Input And Output Contract [#input-and-output-contract]

```json
{
  "goal": "implementation plan for auth",
  "budget": 1200,
  "at": "HEAD"
}
```

* JSON output includes `goal`, `snapshotSha`, `budget`, `usedTokens`, `selectedHits`, and `hits`.
* `--view minimal` is the safest default for agent context windows.
* `--view default` preserves a balanced amount of detail.
* `--view full` is useful only when downstream reasoning truly needs richer text.

Examples [#examples]

Human terminal flow:

```bash
pnpm ragit context pack "implementation plan for auth" --budget 1200
```

Agent-oriented flow:

```bash
pnpm ragit context pack --input context-pack.json --view minimal --format json
```

Failures And Cautions [#failures-and-cautions]

* Do not mix `--input` with positional `goal`, `--budget`, or `--at`.
* An empty `goal` is rejected.
* This command remains snapshot-bound; it does not include working-memory state unless you explicitly use <a href="/en/docs/commands/memory/recall">`memory recall`</a>.
